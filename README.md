# Placement Management System

A Flask-based web application that helps students prepare for placements.

## Features
- User Authentication
- Resume ATS Analysis
- Placement Prediction (Machine Learning)
- Company Recommendation
- Interview Preparation
- SQL Server Database

## Tech Stack
- Python
- Flask
- SQL Server
- HTML
- CSS
- JavaScript
- scikit-learn

## Installation

git clone <repo>

pip install -r requirements.txt

Configure database/db_connection.py

python app.py
